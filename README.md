Create-A-Kid Genetic Simulator 🧬
An interactive, browser-based genetics tool that generates a cartoon face portrait in real time as students enter genotypes. Designed to accompany the Create-A-Kid Genetic Project assignment, where student pairs combine their "gametes" (coin flips) to produce a child's genotype — then see that child come to life on screen.
No install. No login. No internet required after download. Just open the file and go.

What It Does
Students type genotypes into input boxes (e.g., Bb, ee, WW) and a cartoon face updates instantly to reflect the phenotype. The simulator covers:

25 single-gene traits (Traits A–Z) including complete and incomplete dominance
Polygenic eye color (genes A, B, C — 7 shades from light blue to dark brown)
Polygenic skin tone (genes D, E, F, G — 9 shades from melon to ebony)
A phenotype summary panel listing the child's expressed traits

Visual traits rendered on the face include hair texture, hair color, eye shape, eye size, nose width, lip thickness, freckles, dimples, widow's peak, free vs. attached earlobes, cleft chin, and more.

How to Use
Setup (one time)

Download create-a-kid.html
That's it — no installation needed

In the Classroom

Open the file in any modern web browser (Chrome, Firefox, Edge, Safari)
Have students complete their coin-flip gamete activity and combine alleles with their partner to get the child's genotypes
Students type each genotype into the matching input box
The face portrait updates live as they type
Students can screenshot or describe their child for the family portrait portion of the assignment


Tip: Boxes left empty default to heterozygous, so the face is never blank — great for demos before students have finished filling in all traits.

Works Offline
Once downloaded, the file runs entirely in the browser. No wi-fi needed — safe for Chromebook carts or computer labs with restricted internet access.
